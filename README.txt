	В файле LearningFile.ipynb содержится код, необходимый для обучения сети(рекомендуется запускать через Google Colaboratory). Файл main.py содержить код, который отвечает за распознавание цифр, которые рисует пользователь на специальном холсте. 

В файле model.pickle находится обученная нейросеть. Была выбрана архитектура LeNet5, так как ее проще всего реализовать. Accuracy сети состовляет около 93%, дальше сеть начинает переобучаться. Проблемные цифры 7 и 6. 

Не нужно нажимать на иконку "Развернуть", так как приложение будет работать некорректно. 

Инструкция по установке и запуску:
1. Выполнить команду через Anaconda Promt:
pip install -r requirements.txt

2. Выполнить команду через Anaconda Promt, находясь в директории с файлом main.py:
python main.py

3. Нарисовать цифру и нажать кнопку "Получить число". Чтобы нарисовать новое число, нажмите кнопку "Заново".
